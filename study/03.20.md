# JavaScript 변수선언
- const, let, var가 있다
- var : 자바스크립트 변수선언에 이제 안쓰는것
- var는 변수 중복선언가능, 재할당가능, 선언만가능
- 이런 문제로 이상한곳에 이상한 데이터가 할당이되고 오류가 발생하게됨
- 이 문제로 이제 사용하지 않음

## const
- 변수의 중복선언 불가능
- 변수의 재할당 불가능
- 변수를 선언과 초기화 동시진행 해야함(재할당이 불가능해서)

## let
- 변수의 중복선언 불가능
- 변수의 재할당 가능
- 변수를 선언만 가능(재할당이 가능하기 때문)
